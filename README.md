# Zombie API

## Description

This api is created for storing zombie characters and its equipment.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ yarn build
$ yarn start
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
